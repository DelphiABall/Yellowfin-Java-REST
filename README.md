# Yellowfin-Java-REST
Simple Implementation of the REST API for Yellowfin with Java. 
The example includes transport classes that encapsulate all the security headers and tokens and also enable simple single sign on to work. 
There are a few examples for common tasks, but any other features can be simply created by using the transport mechanism and passing the returned JSON. 

## Getting Started
To get started, please visit https://sandbox.yellowfinbi.com/ and follow the link to register. 
You will then receive an email with credentials to the developer sandbox so you can start playing with pulling the demo content into your Java applications. 

The best place to start is using Single Sign-on to provide a quick link between your application and Yellowfin.
